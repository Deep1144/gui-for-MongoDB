# Gui For MongoDB

## <b>Functionalities : </b>

- Update data
- Delete data
- Search field wise (i.e: if you are searching a number or a date you can give two values to search from in between)
- Object views
- Pagination
- Export Database (csv format)
- Export collection (json or csv format )
- Import Collections
